# peterlewinski.com

Personal portfolio website for Dr. Peter Lewinski, EMBA.

## Stack
- **Framework:** Astro 6.x (zero JS by default)
- **Styling:** Tailwind CSS + custom dark luxury theme
- **Animation:** Motion (React islands, loaded on scroll)
- **Fonts:** Cormorant Garamond + DM Sans (self-hosted)
- **Hosting:** Vercel

## Getting Started
```bash
npm install
npm run dev     # http://localhost:4321
npm run build   # Static output to dist/
```
